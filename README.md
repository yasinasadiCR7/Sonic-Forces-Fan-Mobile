# Sonic Forces Fan Mobile Project

A fan-made community project focused on exploring mobile compatibility for Sonic Forces.

## About
This is an unofficial fan project.
It is not affiliated with SEGA.

The goal is to research and develop a way to run the game on mobile devices.

## Goals
- Technical research
- Android compatibility
- Performance optimization
- Mobile controls
- Community collaboration

## Current Status
Early planning stage.
